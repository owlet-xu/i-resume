# 1. 介绍

基于vue+vuex+h5+typescript的个人简历模板。

阿里云部署地址：http://121.196.145.103:31103/#/

支持电脑和手机访问，自适应布局；

支持下载pdf格式文件；

# 2. 如何开始

项目根目录下，`cmd`运行：

```shell
npm install
npm run start
```
> 运行成功后，浏览器输入指定地址（默认[http://localhost:8080/](http://localhost:8080/)），进入登录页面
